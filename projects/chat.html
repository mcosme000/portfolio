About this project

This is a chat app that I made for fun and to practice some DOM manipulation and CSS.

You can send new messages to the different dogs, and even photos!

Maybe the most challenging part of this project was to switch between chats and "identify" the selected one and display the sent messages.

    Some features I would like to improve:

  • - Make the chat responsive
  • - Add new functionalities such as an option to edit the profile, see information for each contact and make fake calls
