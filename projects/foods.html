About this project

A simple site that uses an API to help users search for recipes. It shows basic information such as photo, title, list of ingredients and the link to the original recipe.

The most challenging part was to handle the different overlays I created. When you first visit the site, you will see a landing page, and after you call the API that overlay will stay hidden. I thought it would be easier to handle, but for some reason this part took me a lot of time.
